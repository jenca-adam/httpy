A lightweight socket-based library to create HTTP(s) and WebSocket connections.
## Install
From PyPI
```
   $ pip install httpy
```
On Arch Linux
```
   $ yay -S httpy
```
## Features
   * Cookies support
   * Caching support
   * Easy debugging
   * HTTP authentication
   * Form support
   * JSON support
   * Connection pooling
   * Session support	
   * Runs on PyPy


Docs at <https://httpy.readthedocs.io/>
